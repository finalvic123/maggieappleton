## Module Illustrations

The course is split into 9 modules, each focusing on a specific skill set like _Modern JavaScript Essentials_, _API Requests,_ and _Context for Application State_. 

Each needed their own illustration, so I developed a unified shape language and colour palette to generate a variety of matching ones.
